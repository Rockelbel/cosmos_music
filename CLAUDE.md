# COSMOS 项目协作指南

## 核心原则

### 第一性原理思考

请使用第一性原理思考。你不能总是假设我非常清楚自己想要什么和该怎么得到。请保持审慎，从原始需求和问题出发，如果动机和目标不清晰，停下来和我讨论。如果目标清晰但是路径不是最短，告诉我，并且建议更好的办法。

## 项目背景

COSMOS 是一个沉浸式的音乐播放器，采用宇宙星空主题设计。

## 技术栈

- 纯 HTML/CSS/JavaScript
- Three.js 星空背景
- Netease Cloud Music API

## 设计方向

- 工业未来主义风格
- 有机变形的视觉元素
- 流畅的交互体验
